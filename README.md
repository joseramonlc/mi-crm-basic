# NexusCRM — El CRM del Futuro

Landing page moderna de un CRM todo-en-uno para negocios digitales, construida en HTML/CSS puro con diseño 2026.

🌐 **Demo en vivo:** [joseramonlc.github.io/mi-crm-basic](https://joseramonlc.github.io/mi-crm-basic/)

---

## Características

- **Gestión de Clientes 360°** — Perfiles completos, lead scoring con IA y segmentación dinámica
- **E-Commerce Digital** — Venta de productos digitales sin comisiones (cursos, ebooks, suscripciones)
- **Social CRM** — Captación de leads desde Instagram, TikTok, LinkedIn, Facebook y X
- **Asistente IA** — Copiloto comercial que redacta emails, predice cierres y detecta churn
- **Email Marketing** — Secuencias automáticas, A/B testing, hasta 1M emails/mes
- **Live Chat & Chatbot IA** — Bot entrenado con tu base de conocimiento + WhatsApp API
- **Automatización de Flujos** — Workflows visuales no-code con +200 plantillas
- **Analytics & BI** — Dashboards en tiempo real, cohortes, LTV y alertas IA
- **Facturación** — Facturas automáticas compatibles con España y LATAM
- **Landing Pages** — Constructor drag-and-drop con A/B testing integrado

## Tecnologías

- HTML5 semántico
- CSS3 puro (variables, grid, flexbox, animaciones, glassmorphism)
- Sin dependencias ni frameworks externos

## Uso

Abre `index.html` directamente en cualquier navegador. No requiere servidor.

```bash
git clone https://github.com/joseramonlc/mi-crm-basic.git
cd mi-crm-basic
open index.html
```
